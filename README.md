# PDF Annotation App (Swift)

A personal iOS project built from scratch in **Swift**, enabling users to read and annotate PDFs with highlights, notes, and drawings. Designed with a clean and intuitive UI for seamless PDF interaction on iPhone and iPad.

## Key Features

- View PDFs in **portrait mode (single page)** or **landscape mode (two-page view)**.  
- **Page turning animations** with curl effects for a natural reading experience.  
- Highlight text, add notes, and draw freehand directly on the PDF.  
- Support for **bidirectional and multilingual PDFs**: fully compatible with Arabic, Hebrew, and other right-to-left or mixed-direction documents.  

## Tech Skills Demonstrated

- iOS app development with **Swift**, **UIKit**, and **SwiftUI**  
- PDF handling and annotation using **PDFKit**  
- UI/UX design optimized for mobile reading and annotation  
- Multilingual text support
